## 💡 핵심 개념

### Globs 패턴

| 패턴 | 의미 | 예시 |
|------|------|------|
| `*` | 임의의 문자열 | `*.js` = 모든 JS 파일 |
| `**` | 모든 하위 디렉토리 | `**/*.test.js` = 모든 테스트 파일 |
| `?` | 단일 문자 | `?.js` = 한 글자 이름의 JS 파일 |
| `[abc]` | 문자 집합 | `[abc].js` = a.js, b.js, c.js |

### .mdc 파일

- **확장자**: `.mdc` (Markdown Context)
- **위치**: 프로젝트 루트 또는 폴더 내
- **명명**: `패턴.mdc` (예: `*.test.js.mdc`)
- **적용**: 패턴과 일치하는 파일에만 적용
