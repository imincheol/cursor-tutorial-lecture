# ⌨️ Cursor 단축키 치트시트

> macOS 기준 | Windows/Linux는 `Cmd` → `Ctrl`로 대체

---

## 🔥 가장 많이 쓰는 단축키 TOP 10

| 순위 | 단축키 | 기능 |
|------|--------|------|
| 1 | `Cmd+L` | Agent/Chat 열기 |
| 2 | `Cmd+K` | 인라인 편집 |
| 3 | `Tab` | AI 자동완성 수락 |
| 4 | `Cmd+Shift+L` | 선택 영역을 Chat에 추가 |
| 5 | `Cmd+I` | Quick Chat (빠른 질문) |
| 6 | `Cmd+E` | Background Agent로 전송 |
| 7 | `Cmd+/` | 모델 변경 |
| 8 | `Escape` | 제안 거부 / 닫기 |
| 9 | `Cmd+Enter` | 전체 파일 편집 (Cmd+K에서) |
| 10 | `Cmd+.` | Quick Fix |

## 💡 팁

### 단축키 커스터마이징
```
Cmd+Shift+P → "Preferences: Open Keyboard Shortcuts"
```

### 단축키 PDF 보기
```
Cmd+Shift+P → "Help: Keyboard Shortcuts Reference"
```

### 자주 쓰는 패턴

1. **빠른 리팩토링**
   ```
   코드 선택 → Cmd+K → "리팩토링해줘"
   ```

2. **컨텍스트와 함께 질문**
   ```
   코드 선택 → Cmd+Shift+L → Cmd+L → 질문
   ```

3. **멀티파일 작업**
   ```
   Cmd+L → @파일1 @파일2 → 요청
   ```
