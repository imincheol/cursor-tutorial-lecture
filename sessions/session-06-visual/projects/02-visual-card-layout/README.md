# Project 2: Visual Editor로 카드 레이아웃 수정

## 🎯 학습 목표

- 복잡한 레이아웃을 Visual Editor로 수정
- 그리드/플렉스 레이아웃 조정
- 여러 요소를 동시에 선택하여 수정


---

## 📁 프로젝트 구조

```
02-visual-card-layout/
├── README.md           # 이 파일
├── index.html          # 카드 레이아웃 페이지
└── styles.css          # 스타일시트
```

---

## 🎨 실습 시나리오

여러 개의 카드가 그리드 레이아웃으로 배치되어 있습니다. 특정 카드의 레이아웃을 수정해야 합니다.

**복잡한 요소 설명의 어려움**:
```
"두 번째 줄 왼쪽에 있는 카드의 이미지를 더 크게 만들어줘"
→ Agent가 어떤 카드인지 헷갈림
→ 여러 번 시도
```

**Visual Editor로 간단하게**:
```
1. 카드 클릭
2. "이 카드의 이미지를 더 크게 만들어줘"
→ 정확히 선택된 카드만 수정
```

---

## 🚀 실습 단계

### Step 1: 페이지 열기

1. `index.html` 파일을 브라우저에서 열기
2. 카드 레이아웃이 보이는지 확인

---

### Step 2: Visual Editor로 카드 선택

1. Cursor Agent에서 **Visual Editor** 활성화
2. 수정하고 싶은 카드를 **클릭**
3. 다음 작업들을 시도:

**카드 배경색 변경**:
```
이 카드의 배경색을 연한 파란색으로 바꿔줘
```

**카드 크기 조정**:
```
이 카드를 더 크게 만들어줘
```

**카드 그림자 추가**:
```
이 카드에 부드러운 그림자 효과 추가해줘
```

---

### Step 3: 레이아웃 수정

Visual Editor로 다음 레이아웃 작업들을 시도하세요:

1. **그리드 간격 조정**:
   - 카드 컨테이너 클릭
   - "카드 사이 간격을 더 넓게 만들어줘"

2. **카드 정렬 변경**:
   - 카드 클릭
   - "이 카드를 중앙 정렬해줘"

3. **반응형 레이아웃**:
   - 카드 컨테이너 클릭
   - "모바일에서는 1열로 보이게 해줘"

---

### Step 4: 복잡한 요소 수정

카드 내부의 특정 요소를 선택하여 수정:

1. **이미지 수정**:
   - 카드 내 이미지 클릭
   - "이 이미지를 둥글게 만들어줘"

2. **텍스트 스타일**:
   - 카드 제목 클릭
   - "이 제목을 더 크고 굵게 만들어줘"

3. **버튼 위치**:
   - 카드 내 버튼 클릭
   - "이 버튼을 카드 하단에 고정해줘"

---

## 💡 핵심 개념

### 복잡한 레이아웃 수정

Visual Editor는 다음과 같은 복잡한 레이아웃 작업에 특히 유용합니다:

| 작업           | 말로 설명 난이도 | Visual Editor |
| -------------- | ---------------- | ------------- |
| **그리드 간격** | 🔴 매우 어려움   | ✅ 클릭만      |
| **특정 카드**   | 🔴 위치 설명 필요 | ✅ 클릭만      |
| **내부 요소**   | 🔴 계층 설명 필요 | ✅ 클릭만      |
| **정렬**        | 🟡 보통          | ✅ 클릭만      |

### Visual Editor의 강점

1. **계층 구조**: 부모/자식 요소를 정확히 선택
2. **복잡한 레이아웃**: 그리드, 플렉스 등 쉽게 수정
3. **시각적 피드백**: 실시간으로 변경 사항 확인

## 🎨 실습 예시

### 카드 스타일 변경

```
Before:
- 흰색 배경
- 작은 그림자
- 기본 간격

Visual Editor로 수정:
1. 카드 클릭
2. "이 카드를 연한 보라색 배경에 큰 그림자로 바꿔줘"

After:
- 연한 보라색 배경
- 큰 그림자
- 눈에 띄는 디자인
```

### 레이아웃 재구성

```
Before:
- 3열 그리드
- 좁은 간격

Visual Editor로 수정:
1. 카드 컨테이너 클릭
2. "2열로 바꾸고 간격을 넓게 만들어줘"

After:
- 2열 그리드
- 넓은 간격
- 여유로운 레이아웃
```
