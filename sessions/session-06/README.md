# 6장: Git Worktree - 독립 환경 구성

> **하나의 저장소에서 독립적인 작업 환경을 여러 개 만들 수 있습니다**

## 📝 강의 개요

안녕하세요. 이번 장에서는 **Git Worktree**를 배웁니다.

여러 가지 접근 방법을 동시에 시도하고 싶을 때, 한 환경에서 여러 시도를 하면 컨텍스트가 오염됩니다. Worktree를 사용하면 독립적인 환경을 만들 수 있습니다.

**학습 목표**:
- Git Worktree 기본 사용법
- 독립 환경에서 실험
- 컨텍스트 오염 방지

**예상 소요 시간**: 30분

---

## 🎯 단일 환경 vs Worktree

### 단일 환경

```
main 브랜치에서 A 방법 시도
→ 실패
→ B 방법 시도
→ 컨텍스트 오염 ❌
```

### Worktree

```
worktree-a: A 방법 시도
worktree-b: B 방법 시도
→ 독립적으로 실험 ✅
→ 컨텍스트 오염 없음 ✅
```
