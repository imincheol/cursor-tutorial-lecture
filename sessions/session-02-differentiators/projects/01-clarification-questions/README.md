# Project 1: Clarification Questions 체험

> **Agent 모드 vs Plan 모드의 차이를 직접 체험합니다**

## 🎯 학습 목표

- Agent 모드와 Plan 모드에서 Clarification Questions의 차이 이해
- Plan 모드의 UI 화면 체험
- 요구사항 명확화의 중요성 인식

---

## 📝 실습 개요

같은 요청을 Agent 모드와 Plan 모드에서 각각 시도하여 차이를 비교합니다.

---

## 🚀 실습 단계

### 1단계: Agent 모드에서 시도

1. **Agent 모드 열기**: `Cmd/Ctrl + L`
2. **모드 확인**: 상단에서 "Agent" 모드인지 확인
3. **요청하기**:

```
사용자 인증 시스템을 추가해줘
```

4. **결과 관찰**:
   - Agent가 텍스트로 질문을 던집니다
   - 예: "어떤 인증 방식을 원하시나요? JWT? 세션?"
   - 대화가 끝나고 **다시 타이핑**해야 합니다

### 2단계: Plan 모드에서 시도

1. **Plan 모드 전환**: 상단 드롭다운에서 "Plan" 선택
2. **같은 요청하기**:

```
사용자 인증 시스템을 추가해줘
```

3. **결과 관찰**:
   - **Clarification Questions UI 화면**이 나타납니다!
   - 체크박스나 라디오 버튼으로 선택지 제공
   - **클릭만**으로 선택 가능
   - [Continue] 버튼으로 진행

### 3단계: 차이 비교

| 항목 | Agent 모드 | Plan 모드 |
|------|-----------|-----------|
| 질문 형식 | 텍스트 | **UI 화면** ✅ |
| 답변 방식 | 타이핑 | **클릭** ✅ |
| 편의성 | 불편 | **편리** ✅ |
| 선택지 | 없음 | **명확** ✅ |

## 💡 실습 예시

### Agent 모드 (텍스트 질문)

```
You: 사용자 인증 시스템을 추가해줘

Agent: 몇 가지 확인이 필요합니다:
1. 인증 방식은 JWT와 세션 중 어느 것을 선호하시나요?
2. 소셜 로그인을 포함하시겠습니까?
3. 이메일 인증이 필요한가요?

[대화 끝]

You: [다시 타이핑] JWT로 해주고, Google 소셜 로그인 포함, 이메일 인증 필요해
```

### Plan 모드 (UI 화면)

```
You: 사용자 인증 시스템을 추가해줘

[Clarification Questions UI 화면]

인증 방식을 선택하세요:
○ JWT 인증 ← 클릭
○ 세션 기반 인증

소셜 로그인:
☑ Google ← 클릭
□ GitHub
□ 포함 안 함

추가 기능:
☑ 이메일 인증 ← 클릭
□ 2단계 인증

[Continue] ← 클릭
```

**결과**: Plan 모드가 훨씬 빠르고 편리합니다!

## 🎓 학습 포인트

### 1. Plan 모드의 장점

- **빠른 선택**: 타이핑 대신 클릭
- **명확한 옵션**: 선택지가 시각적으로 제공됨
- **실수 방지**: 오타나 잘못된 입력 없음
- **효율적**: 시간 절약

### 2. 언제 Plan 모드를 사용할까?

✅ **Plan 모드 추천**:
- 복잡한 기능 구현
- 여러 선택지가 있는 경우
- 요구사항이 불명확한 경우
- 계획을 먼저 검토하고 싶을 때

❌ **Agent 모드로 충분**:
- 간단한 수정
- 요구사항이 명확한 경우
- 빠른 대화가 필요한 경우

### 3. Clarification Questions 활용 팁

```bash
# 수동으로 명확화 요청
You: /clarify 이 기능의 요구사항을 정리해줘

# 구체적으로 질문
You: 인증 시스템을 추가해줘. 
     /clarify 어떤 옵션들이 있는지 보여줘
```
