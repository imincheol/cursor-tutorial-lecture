## 🔍 Past Chat 참조란?

Cursor에서 **이전 대화 내용**을 현재 대화에서 참조할 수 있는 기능입니다.

```
채팅 입력창:
@past-chat "로그인 기능 구현" 내용을 참고해서 회원가입도 만들어줘
```

### 주요 기능

- 과거 대화 검색 및 참조
- 이전 맥락을 새 대화에 연결
- 반복 설명 불필요
- 일관된 코드 스타일 유지

### 사용 방법

1. 새 대화 시작 (Cmd/Ctrl + L)
2. `@past-chat "검색어"` 입력
3. 이전 대화 내용을 참조하여 요청
4. AI가 이전 패턴을 따라 코드 생성

## 🚀 데모: 다국어 주석 추가

### 실습 목표

Past Chat 참조 기능을 사용하여 일관된 스타일로 다국어 주석을 추가합니다.

### Step 1: 첫 번째 대화 - 한국어 주석

`demo-multilang/demo.js` 파일을 열고 Cursor Agent에게 요청:

```
이 함수에 한국어 주석을 추가해줘.
각 주요 단계마다 설명을 달아줘.
```

**결과**: 한국어 주석이 추가됩니다.

### Step 2: 두 번째 대화 - 일본어 주석

새 대화를 시작하고 요청:

```
이 함수에 일본어 주석을 추가해줘.
한국어 주석과 같은 스타일로.
```

**결과**: 일본어 주석이 추가됩니다.

### Step 3: Past Chat 참조 - 베트남어와 영어

**핵심 단계!** 새 대화에서 Past Chat 참조:

```
@past-chat "한국어 주석" 과 "일본어 주석" 대화를 참고해서
이 함수에 베트남어와 영어 주석도 추가해줘.
같은 스타일로 각 언어별로 주석을 달아줘.
```

**결과**: AI가 이전 대화의 패턴을 참고하여 일관된 스타일로 베트남어와 영어 주석을 추가합니다!

### 학습 포인트

✅ **Past Chat의 장점**:
- 이전 대화의 스타일과 패턴을 재사용
- 일관된 코드 스타일 유지
- 반복 설명 불필요

✅ **@past-chat 사용법**:
- `@past-chat "검색어"`: 이전 대화 검색
- 검색어는 대화 내용의 키워드
- 여러 대화를 동시에 참조 가능

자세한 내용은 `demo-multilang/instructions.md`를 참고하세요.

## 💡 추가 팁

### Past Chat을 효과적으로 사용하는 방법:

1. **명확한 검색어**: 대화 내용의 핵심 키워드 사용
2. **구체적인 요청**: "같은 패턴으로", "같은 형식으로" 명시
3. **여러 대화 참조**: 필요시 여러 대화를 동시에 참조
4. **대화 제목**: 중요한 대화는 의미있는 제목으로 저장

### 일반적인 활용 사례:

- 🔄 **API 패턴**: 로그인 → 회원가입 → 비밀번호 재설정
- 🎨 **디자인 시스템**: Primary → Secondary → Danger 버튼
- 🧪 **테스트 패턴**: 성공 케이스 → 실패 케이스 → 에러 처리
- 📝 **문서화**: API 문서 → 가이드 → README
- 🌐 **다국어**: 한국어 → 영어 → 일본어 → 베트남어

### 주의사항:

- ⚠️ Past Chat은 대화 내용만 참조 (파일 내용 아님)
- ⚠️ 너무 오래된 대화는 찾기 어려울 수 있음
- ⚠️ 검색어가 너무 일반적이면 원하는 대화를 못 찾을 수 있음
