# 교시별 실습 가이드

## 📖 전체 스토리

```
당신은 Copilot 사용자였다.
AI가 말을 안 듣는 문제로 고민했다.
Cursor라는 이름을 듣고 찾아왔다.

이 강의는 그 여정의 기록이다.
```

## 📚 강의 구성

### 정규 강의 (1-4교시)

> **Cursor의 핵심을 발견하는 여정**

```
sessions/
├── session-01/  # 1교시: Hooks 발견 (4개 프로젝트)
├── session-02/  # 2교시: Debug & Visual (4개 프로젝트)
├── session-03/  # 3교시: CLI Agent (2개 프로젝트)
└── session-04/  # 4교시: Multi-Agent (3개 프로젝트)
```

**총 13개 프로젝트, 4시간**

### 추가 강의 (5-6교시)

> **실무에서 바로 적용하는 자동화**

```
sessions/
├── session-05/  # 5교시: 반복 자동화 (3개 프로젝트)
├── session-06/  # 6교시: 품질 자동화 (3개 프로젝트)
└── BONUS_...    # 보너스: Nightly 기능
```

**총 6개 프로젝트, 2시간**
