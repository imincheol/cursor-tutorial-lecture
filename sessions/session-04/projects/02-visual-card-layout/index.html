카드 레이아웃 수정 실습

Visual Editor를 사용하여 카드를 클릭하고 레이아웃을 수정해보세요!

프로젝트 1

이 카드의 스타일을 Visual Editor로 수정해보세요.

프로젝트 2

카드를 클릭하여 배경색이나 그림자를 변경할 수 있습니다.

프로젝트 3

이미지를 클릭하여 크기나 모양을 조정할 수 있습니다.

프로젝트 4

제목이나 텍스트를 클릭하여 스타일을 변경할 수 있습니다.

프로젝트 5

버튼을 클릭하여 위치나 스타일을 조정할 수 있습니다.

프로젝트 6

카드 컨테이너를 클릭하여 그리드 레이아웃을 변경할 수 있습니다.

실습 방법

  1. Cursor Agent에서 Visual Editor 활성화
  2. 수정하고 싶은 카드 또는 요소를 클릭
  3. Agent에게 스타일 또는 레이아웃 수정 요청
  4. 예시:
    • "이 카드의 배경색을 연한 파란색으로 바꿔줘"
    • "이 이미지를 둥글게 만들어줘"
    • "카드 사이 간격을 더 넓게 만들어줘"
