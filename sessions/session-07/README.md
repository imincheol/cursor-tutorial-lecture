---

## 🎯 단일 Agent vs Multi-Agent

### 단일 Agent

```
1. 기능 개발 (IDE)
2. 테스트 작성 (IDE)
3. 문서 작성 (IDE)
→ 순차적 ⏱️
```

### Multi-Agent

```
동시에:
- IDE Agent: 기능 개발
- CLI Agent 1: 테스트 작성
- CLI Agent 2: 문서 작성
→ 병렬 ⚡
```
